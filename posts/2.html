环境准备

win 10 vscode github账号

一、nodejs安装

打开nodejs官网，https://nodejs.org/en/download/，下载自己喜欢的版本，建议下载16以下的版本。历史版本链接：https://nodejs.org/en/download/releases/。

安装过程请参考度娘。

按下win+R,输入cmd,在cmd窗口输入 node -v、 npm -v，正常的话会显示版本号。

修改npm源（可选）

npm config set registry https://registry.npm.taobao.org

二、git安装

打开git淘宝开源镜像网站：https://registry.npmmirror.com/binary.html?path=git-for-windows/v2.36.1.windows.1/，选择自己合适的版本。

安装过程请参考度娘。
